LAB 3 — CLASS STRUCTURE

Car Template

public class Car {
  private int door, window;
  public Car() { door=0; window=0; }
  public Car(int dd, int ww) { door=dd; window=ww; }
  public void display() {
    System.out.println("doors: "+door);
    System.out.println("windows: "+window);
  }
  public int getDoor() { return door; }
  public void setDoor(int door) { this.door=door; }
  public int summation() { return door+window; }
}

Rectangle Methods

public double computeArea() {
  return length*width;
}
public double computePerimeter() {
  return 2*(length+width);
}

Use this.x=x when param name = field name.

LAB 4 — ARRAY INSIDE A CLASS

Add

if(nbm < arrMarks.length) {
  arrMarks[nbm]=mark; nbm++;
  return true;
} else return false;

Delete (swap trick)

arrMarks[pos]=arrMarks[nbm-1];
arrMarks[nbm-1]=0;  // 0 for int!
nbm--;

Search

for(i=0;i<nbm;i++)
  if(arrMarks[i]==mark) return i;
return -1;

Average

return (double)sum/nbm;

Cast! Or you lose decimals.

LAB 5 — ARRAY OF OBJECTS

Delete (null not 0)

ArrEmp[pos]=ArrEmp[nbe-1];
ArrEmp[nbe-1]=null;
nbe--;

Oldest

Employee old=ArrEmp[0];
for(i=1;i<nbe;i++)
  if(ArrEmp[i].getAge()>old.getAge())
    old=ArrEmp[i];
return old;
LAB 6 — AGGREGATION

splitGuests

int j=0,k=0;
for(i=0;i<nbGuests;i++) {
  if(arr[i].getNationality().equals("Saudi"))
    saudi[j++]=arr[i];
  else expat[k++]=arr[i];
}
return k;

5 Critical Rules

• Strings → .equals() not ==
• Object slots → null not 0
• Loop to nbe/nbm not array.length
• Average → cast: (double)sum/n
• Setter same name → need this.
